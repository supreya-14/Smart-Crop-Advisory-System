# 🌱 Smart Crop Advisory System

A full-stack web application that provides AI-powered crop recommendations and weather-based farming advice for farmers.

## 🚀 Features

- 👨‍🌾 Farmer Registration & Login
- 🌾 AI-Based Crop Recommendation
- 🤖 Gemini AI Chatbot
- 🌦️ Weather Forecast Integration
- 📊 Crop & Soil Information
- 🔐 JWT Authentication
- 📱 Responsive User Interface

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

### APIs
- Google Gemini API
- OpenWeatherMap API

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/supreya-14/Smart-Crop-Advisory-System.git
```

### Backend

```bash
cd smart-crop-advisory
cd backend
npm install
npm start
```

### Frontend

```bash
cd smart-crop-advisory
cd frontend
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file inside the `backend` folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
JWT_EXPIRES_IN=7d
GEMINI_API_KEY=your_gemini_api_key
WEATHER_API_KEY=your_openweathermap_api_key
CLIENT_URL=http://localhost:5173
```
